## Product Positioning

- Category: Philosophy quotes + mindfulness — NOT generic motivation
- Unique mechanic: 4-direction swipe (UP/RIGHT/LEFT/DOWN = stoicism/discipline/reflection/philosophy)
- Unique feature: Philosophy Map (visual preference profile)
- Bilingual: EN + ES (full parity)
- Pricing: Free → Trial (7d) → Inside $4.99 lifetime | Packs $2.99 each
- No account required (anonymous UUID auth)

## Target Audience

- Primary: 25-44, intellectually curious, values depth
- Secondary: Philosophy students, Stoicism community, mindfulness seekers
- NOT: Casual motivational quote consumers

## Competitor Landscape

- Stoic (app): daily Stoic quotes, large user base
- Daily Stoic: lifestyle brand + app
- Waking Up: meditation + some philosophy (premium, expensive)
- Generic quote apps: too broad, not our competition

## MindScrolling's Differentiated Angles

1. 4-direction swipe maps to philosophy currents (unique mechanic)
2. Philosophy Map = visual identity for the user (not in any competitor)
3. Pack deep-dives per philosophical current (curriculum-like)
4. Anonymous — no signup friction
5. Bilingual EN/ES from day one
6. Daily Challenges (habit formation without gamification pressure)
