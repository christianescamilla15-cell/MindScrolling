El usuario está liderando el desarrollo de MindScrolling, una app React de frases filosóficas con navegación por swipe.

**Rol:** Product owner + coordinador técnico. Gestiona múltiples agentes especializados (PO, API Architect, Backend Implementer, QA Reviewer) y usa el Scrum Coordinator para consolidar sus outputs.

**Preferencias de trabajo:**
- Prefiere backlogs concretos y accionables con criterios de aceptación en una línea
- Quiere evitar complejidad innecesaria — foco en MVP
- Trabajo organizado por fases (sprints) con responsables claros
- Decisiones de producto pendientes deben ser explícitas y visibles, no asumidas

**Stack del proyecto:** React (JSX), Node.js + Fastify + PostgreSQL (Supabase) + Railway
