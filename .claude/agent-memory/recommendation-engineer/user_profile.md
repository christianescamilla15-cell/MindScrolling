The user is the founder and product owner of MindScrolling. Has a strong vision for the app as "TikTok for philosophy." Understands the concept of personalization and feed algorithms at a product level, not necessarily the math behind them.

Trusts the Recommendation Engineer to own the algorithm completely. Wants to understand changes in terms of user impact (will users see more relevant content?) rather than mathematical notation. Always approve weight changes or model changes with a brief explanation of why it improves the experience.

Does not want to be asked about implementation details — take ownership and execute. Report results in the structured output format from the agent spec.
